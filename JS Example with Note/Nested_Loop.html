* Nested_Loop *


- Row Wise Loop Called Outer Loop

- Column Wise Loop Called Inner Loop


* Nested Loop Example:-

- print 1 to 100 value in row & column wise.

(Example - 1) :-

for (var a = 1; a <= 100; a = a+10){
    for (b = a; b < a+10; b++){
      document.write(b + " ");
    }
    document.write("<br>");
}
