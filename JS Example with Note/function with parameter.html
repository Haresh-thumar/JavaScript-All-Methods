* function with parameter *

- એક function માં 1 થી વધારે parameter pass કરીએ છીએ અને એને print કરવા માટે એક default value પણ pass કઈ શકીએ છીએ અને પછી concatination કરીને variable pas કરવાનું હોય છે. ઉપર parameter માં આપણે કોઈ પણ default value pass કરીએ તો function declair કરીએ ત્યારે ત્યાં કાંઈ parameter માં variable કે value enter ના કરીએ તો ત્યાં default parameter ની value get કરે છે. જો function declair કરીએ ત્યારે કોઈ પણ variable કે value pass કરવી પડે છે bydefault તે ઉપર declair કરેલી value get કરે છે અને નીચે function માં new value change કરવી હોય તો new value add કરીએ તો new value print કરે છે. જો ઉપર parameter માં ખાલી variable pass કરાવીને નીચે declair કરેલા function માં જો કાંઈ argument pass કરવામાં નહિ આવે તો તેની value undefined print કરશે.



* Simple Parameter structure :- *

function functionName(parameter1, parameter2) {
   statement
}

functionName(argument1, argument2);


(1) Parameter with set default value :-

(Example - 1) :-

function abc(fname ="haresh", lname ="thumar") {
   document.write("hello" + fname + "" + lname);
}

abc();
abc(fname, lname);
abc("ram", + "singh")

(output) :-

hello haresh thumar

hello haresh thumar

hello ram singh


(2) simple Function With Parameter :-

(Example - 2) :-

function abc(a, b) {
   document.write(a + b);
}

abc(20, 30);
abc(40, 50);

(output) :-

50

90


(3) print undefined value :-

(Example - 2) :-

function abc(a, b) {
   document.write("hello" + a + b);
}

abc();

(output) :-

hello undefined undefined
