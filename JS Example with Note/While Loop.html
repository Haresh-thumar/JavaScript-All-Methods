* While loop *

* Loop Devided in 3 Parts :- *

1) initialization

2) condition

3) increment / decrement


1) initialization :-

- iniatialization means એક variable લઇ તેમાં value assign કરવી અને તેને આગળ check કરાવવી (function માં).


2) Condition :-

- condition માં statement check થાય છે. જો condition false થઇ જશે તે loop ની બહાર નીકળી જશે. અને જો condition સાચી પડશે તો એ ત્રીજા part increment / decrement માં જાય છે એ code ઉપર depend છે, increment કરવું તો કેટલું કરવું. increment માંથી value પાછી condition માં જશે. જો false હશે તો loop ની બહાર થઇ જશે અને true હશે તો તે process continue ચાલુ રહેશે.


3) Increment / Decrement :-

- increment means a++++++ value (+ value).

- decrement means a------ value (- value).




* While Loop Syntax (Increment Value):-

- increment means 1 to 10 value print.

(Example - 1) :-

Simple Example

var a = 1;
while(a <= 5) {
   document.write("Hello Haresh");
   a = a + 1; or a++;
}

(output) :-

Hello Haresh

Hello Haresh

Hello Haresh

Hello Haresh

Hello Haresh

(Example - 2) :-

Number with Print Value Example

var a = 1;
while(a <= 5) {
   document.write(a + ") Hello Haresh");
   a = a + 1; or a++;
}

(output) :-

1) Hello Haresh

2) Hello Haresh

3) Hello Haresh

4) Hello Haresh

5) Hello Haresh

(Example - 3) :-

List with Number with Print Value Example

var a = 1;
document.write("<ul>");
while(a <= 5) {
   document.write("<li>" a + ") Hello Haresh </li>");
   a = a + 1; or a++;
}
document.write("</ul>");

(output) :-

  • 1) Hello Haresh
  • 2) Hello Haresh
  • 3) Hello Haresh
  • 4) Hello Haresh
  • 5) Hello Haresh


* While Loop Syntax (Decrement Value):-

- Decrement means 10 to 1 value print. (Reverse Print Value)

(Example - 1) :-

Number with Print Value Example

var a = 5;
while(a >= 1) {
   document.write(a + ") Hello Haresh");
   a = a - 1; or a--;
}

(output) :-

5) Hello Haresh

4) Hello Haresh

3) Hello Haresh

2) Hello Haresh

1) Hello Haresh

(Example - 2) :-

List with Number with Print Value Example

var a = 5;
document.write("<ul>");
while(a >= 1) {
   document.write("<li>" a + ") Hello Haresh </li>");
   a = a - 1; or a--;
}
document.write("</ul>");

(output) :-

  • 5) Hello Haresh
  • 4) Hello Haresh
  • 3) Hello Haresh
  • 2) Hello Haresh
  • 1) Hello Haresh
