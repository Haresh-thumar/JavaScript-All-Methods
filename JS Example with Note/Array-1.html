* Declair Array :-

var ary = [10, 20, 30];

-- simple એક variable માં એક જ value store કરી શકાય છે. જો આપણે એક કરતા વધારે store કરવા જશુ તો error આવી જશે. માટે એક variable માં multiple value store કરવા માટે array નો ઉપયોગ થાય છે.

-- એક class માં ઘણાં બધા student હોય છે, એ બધા student ના roll no. હોય છે. એમ array માં પણ roll no. હોય છે, જે 0 થી ચાલુ થઇ છે. આમાં roll no. ને index કહેવાય છે.



* Simple Array :-

(Example - 1) :-

print numeric value

var haresh = [10,20,30,40,50];
document.write(haresh);

(output) :-

10,20,30,40,50

(Example - 2) :-

print string value

var haresh = ["haresh", "rutvik", "parth", "divyesh", "rushabh"];
document.write(haresh);

(output) :-

haresh, rutvik, parth, divyesh, rushabh


* use multiple data type in array :-

(Example - 1) :-

var haresh = [10,"haresh","google",true,null];

* for loop with list item in Array :-

(Example - 1) :-

for loop with list item in Array (numeric value)

var haresh = [10,20,30,40,50];
document.write("<ul>");
for (a = 0; a <= 4; a++){
    document.write("<li>" + haresh[a] + "</li>");
}
document.write("</ul>");

(output) :-

  • 10
  • 20
  • 30
  • 40
  • 50

(Example - 2) :-

for loop with list item in Array (string value)

* use sum formula with for loop in Array :-

(Example - 1) :-

for loop with list item in Array (numeric value)
