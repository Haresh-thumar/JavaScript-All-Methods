* Conditional ternary operator *

- આ statement if else ની જેમ જ work કરે છે, પણ આમાં એક સાથે multiple condition check નથી શકાતી.



(1) Code:-

(Example - 1) :-

var a = 100;
var = b;

(a == 100) ? b = "true" : b = "false";

document.write(b);

(output) :-

true

(Example - 2) :-

var a = 100;
var = b;

(a == 500) ? b = "true" : b = "false";

document.write(b);

(output) :-

false


(2) Shorthand Code & Add Value :-

(Example - 1) :-

var a = 100;
var = b;

b = (a == 100) ? "true" : "false";

document.write(b);

(output) :-

true

(Example - 2) :-

var a = 100;
var = b;

b = "Value is" + (a == 100 ? "true" : "false");

document.write(b);

(output) :-

value is false
