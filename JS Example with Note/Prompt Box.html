* Prompt Box *

- prompt box એક alert box ની જેમ જ work કરે છે, prompt box માં બે button display થઇ છે જો cancel button click કરતાં prompt box નો dialog close થઇ જાય છે, અને ok button click કરતાં value get કરીને print કરે છે. આ prompt box true અને false ની જેમ જ work કરે છે.



(1) Simple prompt Box :-

(Example - 1) :-

prompt("do you like our website ?");

(output) :-

open simple prompt dialog box


(2) Variable With prompt Box :-

(Example - 1) :-

var a = prompt("do you like our website ?");

   document.write(a);

(output) :-

open prompt dialog box

- prompt dialog box માં જે value enter કરીએ તે display માં print થાય છે.


(3) if / else Conditon With Prompt Box :-

(Example - 1) :-

var per = prompt("do you like our website ?");

if (per >= 80 && per <= 100) {
   document.write("X ia merit");
}
else if (per >= 60 && per < 79) {
   document.write("X ia 1st division");
}
else if (per >= 45 && per < 59) {
   document.write("X ia 2st division");
}
else if (per >= 33 && per < 44) {
   document.write("X ia 3st division");
}
else if (per < 33) {
   document.write("you are fail");
}
else {
   document.write("enter the valid percentage");
}

(output) :-

open prompt dialog box after enter the value than in display print the your value.
