* Camparison operators *

type of Camparison_operators :-
Operators Discription
== Equal to
=== Equal value & Equal type
!= not Equal
!== not Equal value & not Equal type
> Greater than
< Less than
>= Greater than or Equal to
<= Less than or Equal to


(1) Equal to (==) :-

- only check the value in function.

(Example - 1) :-

var a = 20;
var b = 20;
document.write(a == b);

(output) :-

true

(Example - 2) :-

var a = 22;
var b = 20;
document.write(a == b);

(output) :-

false

(Example - 3) :-

var a = 20;
var b = 22;
document.write(a == b);

(output) :-

false


(2) Equal to value & Equal type (===) :-

- check the value & check the data type in function.

(Example - 1) :-

var a = 50;
var b = 50;
document.write(a === b);

(output) :-

true

(Example - 2) :-

var a = "50";
var b = 50;
document.write(a === b);

(output) :-

false

(Example - 3) :-

var a = 50;
var b = ["12", "22"];
document.write(a === b);

(output) :-

false

(Example - 4) :-

var a = 50;
var b = "Haresh";
document.write(a === b);

(output) :-

false

(Example - 5) :-

var a = 20;
var b = {name: 'haresh', age: '23'};
document.write(a === b);

(output) :-

false
