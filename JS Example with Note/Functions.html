* Functions *

- આ statement if else ની જેમ જ work કરે છે, પણ આમાં એક સાથે multiple condition check નથી શકાતી.



(1) Simple Function:-

(Example - 1) :-

function abc() {
   document.write("hello haresh thumar");
}

abc();

(output) :-

hello haresh thumar


(2) How to Call One Function at a Multiple Time :-

(Example - 1) :-

function abc() {
   document.write("hello haresh thumar");
}

abc();
abc();
abc();
abc();
abc();

(output) :-

hello haresh thumar

hello haresh thumar

hello haresh thumar

hello haresh thumar

hello haresh thumar

hello haresh thumar


(3) How to make multiple Function & call Multiple Time :-

(Example - 1) :-

function abc() {
   document.write("hello haresh thumar");
}

function xyz() {
   document.write("hello haresh thumar");
}

abc();
abc();
abc();

xyz();
xyz();
xyz();

(output) :-

hello haresh thumar
