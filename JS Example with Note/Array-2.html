* Another Way to Create Array *

- New Array() Means કે એક Variable, Array ની નવી Bydefault Value Set કરાવીએ છીએ.


* Create Fix & multiple Get Data Type in Array :-

(Example - 1) :-

Create Fix Get Data Type in Array

var abc = new Array(4);
abc[0] = 10;
abc[1] = "haresh";
abc[2] = true;
abc[3] = null;

for (a = 0; a <= 4; a++){
    document.write(abc[a] + "<br>");
}

(output) :-

10

haresh

true

null

(Example - 2) :-

Create Multiple Value in Array

var abc = new Array();
abc[0] = 10;
abc[1] = "haresh";
abc[2] = true;
abc[3] = false;
abc[4] = null;

for (a = 0; a <= 5; a++){
    document.write(abc[a] + "<br>");
}

(output) :-

10

haresh

true

false

null
