* Alert Box *

- output માં સાચી value અને message alert box માં print કરે છે.



(1) Simple Alert Box :-

(Example - 1) :-

alert("this is alert box");

(output) :-

A is greater


(2) Condition With Alert Box :-

(Example - 1) :-

var a = 10;
var b = 20;

if (a > b){
  alert("A is greater");
}
else {
  alert("B smaller");
}

(output) :-

B is greater

(Example - 2) :-

var a = 10;
var b = 20;

if (a > b){
  alert(a);
}
else {
  alert(b);
}

(output) :-

B is greater


(3) Concetination With Value Print in Alert Box :-

(Example - 1) :-

var a = 30;
var b = 20;

if (a > b){
  alert("value is A :" + a);
}
else {
  alert("value is B :" + b);
}

(output) :-

value is A : 30

(Example - 2) :-

var a = 10;
var b = 20;

if (a > b){
  alert(a + b);
}
else {
  alert(a - b);
}

(output) :-

B is greater
