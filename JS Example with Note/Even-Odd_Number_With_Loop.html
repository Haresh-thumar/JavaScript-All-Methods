* Even / Odd Number With Loop *

1) Even Number With Loop :-

- 2, 4, 6, 8, 10 (બેકી સંખ્યા)


1) Odd Number With Loop :-

- 1, 3, 5, 7, 9




* Even Number With Loop :-

(Example - 1) :-

for (var a = 1; a <= 10; a++){
    if(a % 2 == 0){
       document.write(a + "<br>");
    }
}

(output) :-

2

4

6

8

10


* Odd Number With Loop :-

(Example - 1) :-

for (var a = 1; a <= 10; a++){
    if(a % 2 != 0){
       document.write(a + "<br>");
    }
}

(output) :-

1

3

5

7

9
