* Logical Operator *

- if નાં statement માં બે જ condition આવે છે, true & false

- if નાં statement માં condition check કરવા માટે કાંઈક value નું camparison કરવું પડે છે, જેમ કે two value , two variable, multiple values, multiple વરીઅબીલેસ... આની value campare કરવા માટે camparison operator નો use થાય છે.

type of Camparison_operators :-
Operators Discription
&& Logical AND operator
|| Logical OR operator
! Logical NOT operator


(1) Logical AND operator (&&) :-

- First Condition 45 >= 50... 52 is greater than to 50...

- Second Condition 50 <= 55... 50 is less than to 55...

- if statement is true Becouse two condition is the true...

(Example - 1) :-

var haresh = 50;

if(haresh >= 45 && haresh <= 55){
  document.write("yes haresh is the older 50 years...");
};

(output) :-

yes haresh is the older 50 years...


(2) Logical OR operator (||) :-

- First Condition 52 >= 50... 50 is not greater than to 52...

- Second Condition 50 <= 55... 50 is less than to 55...

- if statement is true Becouse one condition is the true...

(Example - 1) :-

var haresh = 50;

if(haresh >= 52 || haresh <= 55){
  document.write("yes haresh is the older 50 years...");
};

(output) :-

yes haresh is the older 50 years...


(3) Logical NOT operator (!) :-

- First Condition 52 >= 50... 50 is not greater than to 52...

- Second Condition 50 <= 55... 50 is less than to 55...

- statement is true Becouse 30 is greater than to 12 but this statement print the reverse value...

(Example - 1) :-

var a = 30;
var b = 15;

if(!a >= 12){
  document.write("yes haresh is the older 50 years...");

(output) :-

none (this operator is reverse value print)
