* global & local varible *



=> Local Variable :-

- global variable નો scope અંદર પણ છે અને બહાર પણ છે. global variable ની અંદર પણ work કરે છે, અને બહાર પણ work કરે છે.

=> Global Variable :-

- local variable નો scope અંદર સુધી જ છે. local variable અંદર જ work કરે છે, બહાર general work નથી કરતો.



Global & Local Variable

(Example - 1) :-

var a = 10;
function hello() {
   document.write(a);
}
hello();

(output) :-

10

(Example - 2) :-

var a = "hello google";

   document.write(a);
}

   document.write(a);

hello google
hello google
