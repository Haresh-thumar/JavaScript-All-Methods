* events *

type of basic events :-
Click Event Mouse Event Key Event Window Event
Click (onclick) Mouse Hover (onmouseenter) Key Press (onkeypress) Load(onload)
Double Click (ondblclick) Mouse Out (onmouseout) Key Up (onkeyup) Unload (onunload)
Right Click (oncontextmenu) Mouse Down (onmousedown) Resize (onresize)
Mouse Up (onmouseup) Scroll (onscroll)
