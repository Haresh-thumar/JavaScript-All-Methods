* Break & Continue Statement *

* Break Statement :-

- એક for loop 5 time ફરે છે જયારે if ની condition માં variable ની નવી value add કરીએ અને break condition નો use કરશુ તો statement ની condition new value સુધી print થશે. પછી તે statement for loop ની બહાર નીકળી જશે.


* Continue Statement :-

- એક for loop 5 time ફરે છે જયારે if ની condition માં variable ની નવી value add કરીએ અને continue condition નો use કરશુ તો statement ની condition new value હશે તે update થઈને ત્યાં if condition નું statement print થશે. અને પછી for loop ચાલુ રહેશે. ને બધી value print થશે.




1) Break Statement :-

(Example - 1) :-

- without continue & break statement value.

for (var a = 1; a <= 5; a++){
    if (a == 3){
       document.write("hey : " + a + "<br>");
    }
    document.write("number : " + a + "<br>");
}

(output) :-

number : 1

number : 2

hey : 3

number : 3

number : 4

number : 5

(Example - 1) :-

- break statement value.

for (var a = 1; a <= 5; a++){
    if (a == 3){
       document.write("hey : " + a + "<br>");
       break;
    }
    document.write("number : " + a + "<br>");
}

(output) :-

number : 1

number : 2

hey : 3


2) Continue Statement :-

(Example - 1) :-

- without continue & break statement value.

for (var a = 1; a <= 5; a++){
    if (a == 3){
       document.write("hey : " + a + "<br>");
    }
    document.write("number : " + a + "<br>");
}

(output) :-

number : 1

number : 2

hey : 3

number : 3

number : 4

number : 5

(Example - 1) :-

- continue statement value.

for (var a = 1; a <= 5; a++){
    if (a == 3){
       document.write("hey : " + a + "<br>");
       continue;
    }
    document.write("number : " + a + "<br>");
}

(output) :-

number : 1

number : 2
