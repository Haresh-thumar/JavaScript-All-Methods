* if condition (statement) *

- if નાં statement માં બે જ condition આવે છે, true & false

- if નાં statement માં condition check કરવા માટે કાંઈક value નું camparison કરવું પડે છે, જેમ કે two value , two variable, multiple values, multiple વરીઅબીલેસ... આની value campare કરવા માટે camparison operator નો use થાય છે.


(1) Example - 1

- campare the data value in if statement.

var a = 100;
var b = 100;

if (a == b) {
document.write("A is Greater");
}

(output) :-

A is Greater

var a = 120;
var b = 100;

if (a == b) {
document.write("A is Greater");
}

(output) :-

none (none means do not show data in display)


(2) Example - 2

- campare the data type in if statement.

var a = 100;
var b = 100;

if (a === b) {
document.write("A is Greater");
}

(output) :-

A is Greater

var a = 120;
var b = "100";

if (a === b) {
document.write("A is Greater");
}

(output) :-

none (none means do not show data in display)





NOTE:- જો operator નું camparison true હશે તો statement true થશે અને તો જ data display થશે નહીંતર data display નહિ થાય.
