* For Loop *

* Loop Devided in 3 Parts :- *

- initialization, condition, increment/decrement


1) structure Working :-

- ₹ for loop માં initialization, condition, increment / decrement એક જ લઈને માં લખાય છે. for loop એ while loop & do while loop ની જેમ જ work કરે છે બસ આ loop માં shorthand code માં લખાય છે.




* For Loop Syntax (Increment Value):-

- increment means 1 to 10 value print.

(Example - 1) :-

Simple Example

for (var a = 1; a <= 5; a++){
    document.write("hello world<br>");
}

(output) :-

Hello Haresh

Hello Haresh

Hello Haresh

Hello Haresh

Hello Haresh

(Example - 2) :-

Number with Print Value Example

for (var a = 1; a <= 5; a++){
    document.write(a + "hello world<br>");
}

(output) :-

1) Hello Haresh

2) Hello Haresh

3) Hello Haresh

4) Hello Haresh

5) Hello Haresh

(Example - 3) :-

List with Number with Print Value Example

document.write("<ul>");
for (var a = 1; a <= 5; a++){
    document.write("<li>", a + ") Hello Haresh </li>");
}
document.write("</ul>");

(output) :-

  • 1) Hello Haresh
  • 2) Hello Haresh
  • 3) Hello Haresh
  • 4) Hello Haresh
  • 5) Hello Haresh


* For Loop Syntax (Decrement Value):-

- Decrement means 10 to 1 value print. (Reverse Print Value)

(Example - 1) :-

Number with Print Value Example

for (var a = 5; a >= 1; a--){
    document.write(a + "hello world<br>");
}

(output) :-

5) Hello Haresh

4) Hello Haresh

3) Hello Haresh

2) Hello Haresh

1) Hello Haresh

(Example - 2) :-

List with Number with Print Value Example

document.write("<ul>");
for (var a = 5; a >= 1; a--){
    document.write("<li>", a + ") Hello Haresh </li>");
}
document.write("</ul>");

(output) :-

  • 5) Hello Haresh
  • 4) Hello Haresh
  • 3) Hello Haresh
  • 2) Hello Haresh
  • 1) Hello Haresh
