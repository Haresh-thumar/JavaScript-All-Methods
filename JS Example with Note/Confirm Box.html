* Confirm Box *

- confirm box એક alert box ની જેમ જ work કરે છે, confirm box માં બે button display થઇ છે જો cancel button click કરતાં confirm બોક્સ નો dialog close થઇ જાય છે, અને ok button click કરતાં program માં આગળ perform કરી શકીયે છીએ. આ confirm box true અને false ની જેમ જ work કરે છે.



(1) Simple Confirm Box :-

(Example - 1) :-

confirm("do you like our website ?");

(output) :-

open simple dialog box


(2) Variable With Confirm Box :-

(Example - 1) :-

var a = confirm("do you like our website ?");

alert(a);

(output) :-

open dialog box

-- જો ok button click કરતાં true નો બીજો dialog box open થાય છે.

-- અને cancel button click કરતાં false નો બીજો dialog box open થાય છે.


(3) if / else Conditon With Confirm Box :-

(Example - 1) :-

var a = confirm("do you like our website ?");

if (a) {
   alert("thanks");
}else {
   alert("sorry");
}

(output) :-

open dialog box

-- ok button click કરતાં thanks નો બીજો dialog box open થાય છે.

-- અને cancel button click કરતાં sorry નો બીજો dialog box open થાય છે.



NOTE :basically confirm box નો ઉપયોગ button પર event call કરાવવામાં થાય છે, function માં વધારે use થાય છે.
