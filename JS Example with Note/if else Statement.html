* if / else statement *

- જો if statement સાચી હશે તો તે statement print કરશે અને જો condition ખોટી હશે તો else statement print કરશે.



(1) Using Camparison Operator (Check Greater Than value & Less Than value) :-

- First Condition 75 > 30... 75 is greater than to 30...

- Second Condition 30 < 75... 30 is small than to 75...

(Example - 1) :-

var x = 75;

if (x > 30){
  document.write("x is greater");
}
else {
  document.write("x is smaller");
}

(output) :-

x is greater

(Example - 2) :-

var x = 30;

if (x > 75){
  document.write("x is greater");
}
else {
  document.write("x is smaller");
}

(output) :-

x is smaller


(2) Using Camparison Operator (Check Equal Value & Equal Value and Equal Data Type) :-

- First Condition 30 == 30... this is equal value...

- Second Condition "30" == 30... this is equal value but data type is not ...

(Example - 1) :-

var x = 30;

if (x == 30){
  document.write("x is Equal Value");
}
else {
  document.write("x is Not Equal Value");
}

(output) :-

x is Equal Value

(Example - 2) :-

var x = "30";

if (x === 30){
  document.write("x is Equal Value & Equal Data Type");
}
else {
  document.write("x is Not Equal Value & Equal Data Type");
}

(output) :-

x is Not Equal Value & Equal Data Type


(3) Using if / else statement :-

- First statement if is true because gender value "male".

- Second statement if is true because gender value "female".

- use the concatination(કોન્કેટિનેશન) to print value & variable in output...

(Example - 1) :-

var name = 'haresh thumar';
var gender = 'male';

if (gender == 'male') {
  document.write("Hello Mr." + name);
}
else {
  document.write("Hello Miss." + name);
}

(output) :-

Hello Mr. haresh thumar

(Example - 2) :-

var name = 'haresh thumar';
var gender = 'female';

if (gender == 'male') {
  document.write("Hello Mr." + name);
}
else {
  document.write("Hello Miss." + name);
}

(output) :-

Hello Miss haresh thumar
