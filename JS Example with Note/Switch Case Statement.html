* Switch Case Statement *

- આ switch statement if statement ની જેમ જ work કરે છે, switch case statement માં multiple condition ચેક કરી શકાય છે.


(1) Example - 1

(Example - 1) :-

var day = 0;

switch (day) {
case 0:
   document.write("today is monday");
break;
case 1:
   document.write("today is tuesday");
break;
case 2:
   document.write("today is wednesday");
break;
case 3:
   document.write("today is thursday");
break;
case 4:
   document.write("today is friday");
break;
case 5:
   document.write("today is saturday");
break;
case 6:
   document.write("today is sunday");
break;

default:
   document.write("Enter the valid day");
}

(output) :-

today is monday

NOTE: one by one check the condition after print the value...

(2) Example - 2

var age = 18;

switch (true) {
case (age >= 15 && age <= 20):    document.write("you are eligible");
break;
case (age >= 21 && age <= 30):    document.write("you are not eligible");
break;
default:
   document.write("enter the valid day");
}

(output) :-

you are eligible

NOTE: જો true switch case માં value true હશે તો statement બરાબર મળશે...

var age = 18;

switch (false) {
case (age >= 15 && age <= 20):    document.write("you are eligible");
break;
case (age >= 21 && age <= 30):    document.write("you are not eligible");
break;
default:
   document.write("enter the valid day");
}

(output) :-

you are not eligible

NOTE: જો false switch case માં value true હશે તો statement reverse print કરશે...
